## Installation

First download/clone `data_tools` from the repository. From the
terminal:

```bash
git clone https://github.com/Nic-Nic/data_tools.git
cd data_tools
```

Then you can install it by running `setup.py` as follows:

```bash
python setup.py sdist
```

Or using `pip`:

```bash
pip install .
```
